1. rész

Mi az a VS Code?

A Visual Studio Code (VS Code) a Microsoft ingyenes, nyílt forráskódú kódszerkesztője. Könnyűsúlyú, de rendkívül bővíthető — a világ legnépszerűbb fejlesztőeszköze.

🚀

Gyors és könnyű

Másodpercek alatt elindul, nem lassítja a géped. Nem úgy, mint a teljes IDE-k (pl. PyCharm), a VS Code minimális erőforrást használ.

🔌

Bővíthető

Több ezer bővítmény érhető el: Python, Git, AI eszközök, témák és sok más. A Marketplace-en mindent megtalálsz.

🌐

Platformfüggetlen

Windows, macOS és Linux rendszeren is fut. A beállításaid szinkronizálhatók gépek között.

💻

Beépített terminál

Nem kell külön terminált nyitnod — közvetlenül a szerkesztőből futtathatod a parancsaidat és a Python scriptjeidet.

Ajánlott bővítmények

Ezek a bővítmények teszik igazán hatékonnyá a VS Code-ot Python fejlesztéshez.

Az alapvető Python bővítmény. IntelliSense (kódkiegészítés), hibakeresés (debugging), linting, formázás és Jupyter notebook támogatás. Ezt mindenképp telepítsd!
Kódelemző (linter), ami jelzi a hibákat, rossz szokásokat és a nem PEP 8 szabványos kódot. A Ruff a modern, gyorsabb alternatíva. Nagyon hasznos clean code tanulásnál!
Megmutatja, ki és mikor módosította az egyes sorokat (git blame). Hasznos csapatmunkában és a kódtörténet megértéséhez.
Automatikus kódformázó. Mentéskor egységes formátumra hozza a kódodat. Python-hoz a Black Formatter a legjobb választás — konzisztens, vitát nem tűrő formázás.
AI kódolási asszisztens, ami közvetlenül a VS Code-ban fut. Többfájlos szerkesztés, terminálparancsok, Git műveletek — minderről bővebben a következő szekcióban!

2. rész

Mi az a Claude Code?

A Claude Code az Anthropic agentikus kódolási asszisztense. Nem csak javasol — olvas, szerkeszt, futtat és ellenőriz. Olyan, mint egy tapasztalt fejlesztő a képernyőd mellett.

🤖

Agentikus megközelítés

Nem csak válaszol a kérdéseidre — önállóan felfedezi a kódbázist, megtervezi a megoldást, végrehajtja és ellenőrzi.

📝

Többfájlos szerkesztés

Egyszerre több fájlt tud módosítani, megérti az összefüggéseket az importok és függőségek között.

🔒

Engedélykérés

Minden módosítás előtt megkérdezi, hogy jóváhagyod-e. Te irányítasz — a diff nézetben látod a változásokat.

🧰

MCP integráció

Külső eszközökhöz csatlakozhat: adatbázisok, böngésző, Slack, GitHub — a Model Context Protocol segítségével.

Első lépések

Telepítés és beállítás

VS Code bővítmény

Nyisd meg a bővítményeket (⌘⇧X), keresd meg a "Claude Code" bővítményt és telepítsd.

Bejelentkezés

Kattints a Claude Code ikonra és jelentkezz be az Anthropic fiókoddal.

Első kérdés

Írd be: "Mesélj erről a projektről" — Claude feltérképezi a kódbázisodat.

CLAUDE.md létrehozása

Hozz létre egy CLAUDE.md fájlt a projekt gyökerében a konvenciókkal és preferenciákkal.

Képességek

Mit tud a Claude Code?

Fedezd fel a Claude Code legfontosabb funkcióit — kattints a fülekre!

Kódgenerálás és szerkesztés

Claude Code képes új kódot írni, meglévőt módosítani, és akár több fájlban egyszerre refaktorálni. Minden változtatást diff nézetben mutat — te döntöd el, hogy elfogadod-e.

"Írj egy Python függvényt, ami ellenőrzi, hogy egy szám prímszám-e"

Fájlműveletek és @-hivatkozások

A @fájlnév szintaxissal közvetlenül hivatkozhatsz fájlokra. Claude képes olvasni, létrehozni és módosítani fájlokat. Még PDF-eket és képeket is megért!

"@app.py — add hozzá az input validálást a process_data függvényhez"

Terminál parancsok futtatása

Claude képes shell parancsokat futtatni: teszteket indítani, csomagokat telepíteni, build-et futtatni. Az eredményt látja és reagál rá.

"Futtasd le a teszteket és javítsd ki az esetleges hibákat"

Git műveletek

Commitol, brancheket kezel, PR-t nyit a GitHub-on. Automatikusan ír commit üzenetet a változásaid alapján.

"Commitold el a változásokat értelmes üzenettel, majd nyiss egy PR-t"

MCP szerverek (Model Context Protocol)

Külső eszközöket csatlakoztathatsz: adatbázisok (PostgreSQL), böngésző, Slack, GitHub API, és sok más. Nyílt szabvány, bárki írhat MCP szervert.

"Csatlakozz az adatbázishoz és kérdezd le az aktív felhasználókat"

Slash parancsok és skillek

A / karakterrel éred el a parancsokat: /model (modellváltás), /compact (kontextus tömörítés), /memory (memória kezelés). Saját skilleket is létrehozhatsz!

/model — modellváltás  |  /compact — kontextus tömörítés  |  /memory — memória

Hookok — automatizálás

Shell parancsok, amik automatikusan futnak bizonyos eseményekre: fájl szerkesztés előtt/után, munkamenet indulásakor. Például: automatikus formázás minden mentés után.

Pre-tool hook: "ruff format" → minden fájl szerkesztés előtt formáz

Agentikus kódolás — szubágensek

Claude képes alágenseket indítani, amelyek párhuzamosan dolgoznak különböző feladatokon. Minden ágens saját kontextussal rendelkezik és visszajelentést ad.

"Keress rá, hol használjuk a process_data függvényt a kódbázisban"
→ Claude egy Explore ágenst indít a kereséshez

Irányítás

Te döntöd el, mennyire legyen önálló a Claude Code. Három mód közül választhatsz.

🛡

Alapértelmezett (Default)

Claude minden fájlmódosítás és parancs futtatás előtt megkérdezi a jóváhagyásodat. A legbiztonságosabb mód kezdőknek.

Ajánlott kezdőknek

Claude először megtervezi a megoldást és megmutatja neked. Csak a terv elfogadása után kezd el kódot írni.

Komplex feladatokhoz

Claude önállóan dolgozik, automatikusan elfogad minden műveletet. Csak akkor használd, ha megbízol a folyamatban!

Konfiguráció

CLAUDE.md — A projekt memóriája

Hozz létre egy CLAUDE.md fájlt a projekted gyökerében, és Claude mindig emlékezni fog a konvencióidra.

CLAUDE.md
# Projekt konvenciók

## Build parancsok
- Tesztek futtatása: `python -m pytest`
- Linter: `ruff check .`
- Formázás: `ruff format .`

## Kódolási szabályok
- Python 3.12+ szintaxis
- Type hintek minden függvényen
- Docstringek Google stílusban
- Magyar kommentek, angol változónevek

## Architektúra
- src/ — forráskód
- tests/ — tesztek (pytest)
- alembic/ — migrációk

Gyakorlati felhasználás

Néhány valós helyzet, ahol a Claude Code igazán ragyog. Kattints a kártyákra a példa promptokért!

🐛

Hibakeresés (Debugging)

Amikor a kódod hibát dob és nem tudod, miért.

Másold be a hibaüzenetet és kérd Claude segítségét:

TypeError: 'NoneType' object is not subscriptable — javítsd ki ezt a hibát az app.py fájlban
🔨

Refaktorálás

Működő, de csúnya kód átalakítása szebb formára.

Claude megérti a kód struktúráját és képes tisztább formára hozni:

@utils.py — refaktoráld ezt a fájlt: bonts ki kisebb függvényeket és adj hozzá type hinteket
🔬

Tesztek írása

Automatikus tesztgenerálás a kódodhoz.

Claude képes pytest teszteket generálni, amik lefedik a szélsőeseteket is:

Írj unit teszteket a @models.py összes függvényéhez pytest-tel
🎓

Tanulás és megértés

Ismeretlen kód vagy koncepció megértése.

Magyarázd el, mit csinál ez a dekorátor és miért használjuk itt
